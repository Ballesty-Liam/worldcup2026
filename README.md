# World Cup 2026 Draft & Leaderboard

A single-page app for a 12-person World Cup 2026 draft pool with live score tracking, a full match schedule, and a player matchup view.

## Live Site

**https://ballesty-liam.github.io/worldcup2026/**

## Features

- **Draft results** — all 48 teams assigned across 4 pots, revealed during a live ceremony and now locked permanently
- **Live leaderboard** — scores auto-update every hour via the ESPN public API; columns for match points, stage bonuses, goal differential, and total
- **Schedule & Next Up** — all 104 fixtures with local kickoff times; highlights today's games and calls out player vs. player matchups
- **Custom scoring** — group wins (+3), draws (+1), knockout wins (+3), plus stage progression bonuses (R32 +1, R16 +1, QF +2, SF +2, Final +3)
- **Mobile-friendly** — responsive header with scrollable nav strip, horizontally-scrollable leaderboard table
