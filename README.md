# World Cup 2026 Draft & Leaderboard

A single-page app for a 12-person World Cup 2026 draft pool with live score tracking.

## Live Site

**https://ballesty-liam.github.io/worldcup2026/**

## Features

- **Draft ceremony** — randomised team assignments revealed pot-by-pot, all 12 players see their team simultaneously
- **Live leaderboard** — auto-updates every hour using AllSportsApi match results
- **Custom scoring** — group wins, knockout wins, and stage progression bonuses
